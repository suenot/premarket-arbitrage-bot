# [​](https://docs.premarket.me/#premarket-documentation)Premarket Documentation
[​](https://docs.premarket.me/#premarket-documentation)
Premarket is a prediction markets aggregator. The platform combines liquidity and market data across multiple sources and exposes a unified API for events, orderbooks, liquidity routes, positions, arbitrage, wallets, and auth flows.
