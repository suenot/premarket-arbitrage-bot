# Конфигурация

Все параметры настраиваются через `.env` файл в корне проекта.

## Параметры

### Обязательные

| Параметр | Тип | Описание |
|----------|-----|----------|
| `PREMARKET_API_KEY` | string | JWT-токен для Premarket.me API |

### Кошелёк

| Параметр | Тип | Описание |
|----------|-----|----------|
| `ETH_WALLET` | string | Адрес Ethereum-кошелька (0x...) |
| `ETH_SECRET` | string | 12-словная BIP-39 мнемоника из MetaMask |

### Настройки бота

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|-------------|----------|
| `DRY_RUN` | bool | `true` | `true` = мониторинг, `false` = торговля |
| `MIN_PROFIT_PCT` | float | `3.0` | Минимальный profit % |
| `MIN_APR` | float | `10.0` | Минимальная годовая доходность |
| `MIN_DEPTH_USD` | float | `10.0` | Минимальная ликвидность в USD |
| `POLL_INTERVAL` | int | `30` | Интервал опроса API (секунды) |
| `MAX_TRADE_USD` | float | `50.0` | Макс. сумма одной сделки |

## Профили

### Консервативный (default)
```env
MIN_PROFIT_PCT=3
MIN_APR=10
MIN_DEPTH_USD=10
MAX_TRADE_USD=50
```

### Агрессивный
```env
MIN_PROFIT_PCT=1
MIN_APR=5
MIN_DEPTH_USD=5
MAX_TRADE_USD=200
```

## Деривация кошелька

Путь: `m/44'/60'/0'/0/0` (стандарт MetaMask). При запуске бот верифицирует адрес.
